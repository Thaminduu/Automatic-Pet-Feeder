# Automatic Pet Feeder

An IoT-enabled automatic pet feeding system designed to provide scheduled and remote-controlled feeding for pets. The system utilizes an ESP32 microcontroller, MQTT communication, and Node-RED dashboards to automate food dispensing while allowing real-time monitoring and control over a wireless network.

The project was developed to address the challenges of maintaining consistent feeding schedules when pet owners are away from home. By combining embedded systems, wireless communication, and automation technologies, the feeder can dispense food automatically according to predefined schedules or manual commands sent through the dashboard.

This project demonstrates the integration of embedded systems, IoT communication protocols, control systems, and web-based monitoring interfaces.
